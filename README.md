# Building LLM Applications with LLM Zoomcamp

This repository contains my work, experiments and capstone project completed as part of the LLM Zoomcamp program by [DataTalks.Club](https://github.com/DataTalksClub/llm-zoomcamp).

The objective of this repository is to develop practical skills in building Large Language Model (LLM) applications and gain hands-on experience with modern AI engineering workflows.

---

## Topics Covered

Throughout the program, I will explore and implement concepts related to:

- Large Language Models (LLMs)
- Prompt Engineering
- Retrieval-Augmented Generation (RAG)
- Embeddings and Vector Databases
- Open-Source LLMs
- LLM Evaluation and Monitoring
- AI Agents and Workflows
- Production-Ready AI Systems

---

## Why I'm Doing This

I am using this repository to strengthen my practical LLM engineering skills by implementing concepts from scratch, completing hands-on assignments and building a production-oriented capstone project.

The focus is on understanding how modern LLM applications are designed, evaluated, and deployed in real-world environments.

---

## Learning Objectives

By the end of this program, I aim to:

- Understand modern LLM architectures and workflows
- Build retrieval-based AI applications
- Work with embeddings and vector search systems
- Evaluate and monitor LLM performance
- Learn best practices for deploying AI applications
- Develop a complete end-to-end LLM project
